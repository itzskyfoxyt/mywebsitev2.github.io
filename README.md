## 🚀 Deploy Your Own (GitHub Pages)

1. **Fork or clone** this repository:
   ```bash
   git clone https://github.com/yourusername/yourusername.github.io.git
   ```

2. **Edit `index.html`** — Replace placeholder text:
   - `YourName` → your actual name
   - `yourusername` → your GitHub username
   - `hello@yourname.dev` → your email
   - Update projects, skills, and stats to match yours

3. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "feat: initial portfolio setup"
   git push
   ```

4. **Enable GitHub Pages:**
   - Go to **Settings → Pages**
   - Set **Source** to `Deploy from a branch`
   - Choose `main` branch, `/ (root)` folder
   - Hit **Save**

5. **Done!** Your site will be live at `https://yourusername.github.io` in about 60 seconds.

## 🎨 Customisation Guide

| What to change | Where to find it |
|---|---|
| Name & title | `<h1 class="hero-title">` in `index.html` |
| About text | `<p class="about-body">` section |
| Skills/tags | `.skill-tags` blocks |
| Projects | `.project-card` blocks (6 total) |
| Stats (years, projects) | `.stat-num` elements in hero |
| Social links | Footer & contact section `href` values |
| Color accent | `--accent` CSS variable (currently `#c8f03e`) |
| Font style | Google Fonts import in `<head>` |
